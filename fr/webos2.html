Choisis votre language, pays, et la timezone de votre site (voir photo). Notez que le réglage de l'horloge est fondamental
Appuyer sur le bouton SETTINGS, et aller dans le and go to the advanced menu (see picture)
Go to NETWORK and choose your desired internet connection type (see picture)
WIRED CONNECTION If you are connected to the internet with a RJ45 cable, the following menu (Wired Connection) will show the name of the network. If you are setup with a DHCP network, let it run automatically. Otherwise, click on MODIFY, uncheck DEFINE AUTOMATICALLY, and enter the required data (Ip adress, gateway, etc etc).
WIFI CONNECTION Select WIFI CONNECTION. Choose the network (see picture), and enter its password.
WARNING: You won't be able to install our player if you do not have an internet connection.
Open the hidden menu by pressing on the SETTINGS button until until a bubble appears on the top right corner Quickly press 4 times the button 0, and press OK.
Adjust the following parameters AUTO POWER = OFF SPEAKER = OFF
Go to APPLICATIONS and uncheck all cases (see picture).
In the SETTINGS menu, go to Advanced NEtwork Configuration
Adjust the following parameters APPLICATION LAUNCH MODE = LOCAL FULLY QUALIFIED DOMAIN NAME = ON - Enter this URL http://webos.deepidoo.com/dj/ipk Be careful, it won't work if the URL is not correct - Application Type = IPK - Finally, go to Local Applcation Upgrade, choose REMOTE and press OK, then confirm
