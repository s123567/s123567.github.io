  • Configuration du réseau
  • Installation de l'application Deepidoo'

Configurer votre langue, continent, pays, et fuseau horaire.

Sélectionnez OFF pour les deux choix de rotation.

Configurez votre connexion internet, Ethernet ou WiFi

Si vous sélectionnez WiFi: Renseignez le mot de passer et connectez vous.

Réglez les heures d'allumage et d'extinction de l'écran (suivant les horaires du magasin).

Pour configurer votre connexion Internet avec IP Fixes: Appuyez sur la touche SETTINGS puis rendez-vous dans ALL SETTINGS, Puis RESEAU et choisissez le type de connexion utilisé.

Choisissez EDIT et renseignez les informations Réseau (Voir avec le siège) puis confirmez.

Pour accéder au menu d'installation, appuyez en maintenant la touche SETTINGS jusqu’à l'apparition de la bulle. Entrez le code 8080 et pressez OK (Attention: Les chiffres n’apparaissent pas à l’écran).

Paramétrages de l'installation: Sélectionnez « Local » pour « Application Launch Mode » et « On » pour « Fully Qualified Domain Name ».

Renseignez l’adresse suivante: http://webos.deepidoo.com/dj.ipk

Choisissez « IPK » dans « Application type ». Lancez l'application en choisissant « Remonte » pour le paramètre « Local application Upgrade », puis confirmez.

Composez le code fournit par Deepidoo puis pressez la touche OK (Vérifiez que l’adresse qui s’affiche corresponde bien à votre magasin avec de valider)
