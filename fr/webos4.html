Menu précédent
  • Première utilisation de l'écran
  • Configuration du réseau
  • Installation de l'application Deepidoo'
Choisissez QUICK START puis sélectionner la langue correspondante.
Désactiver les rotations de l’écran. Configurer la date et l'heure, en fonction de votre lieu puis sélectionner Terminé.
Régler l’heure d’allumage et d’extinction, en fonction de votre magasin.
Appuyer sur la touche SETTINGS sur la télécommande.
Aller dans le menu Général puis choisir le type de réseau correspondant (Ethernet ou Wifi)
CONNEXION FILAIRE (RJ45) : La connexion se fera automatiquement
CONNEXION WIFI: Sélectionner le réseau correspondant et renseigner le mot de passe.
CONNEXION DHCP: Laisser les paramétres par défaut;
IP FIXES: Sélectionner modifier puis renseigner les informations réseau correspondantes.
Appuyer sur la touche SETTINGS de la télécommande. Sélectionner Configuration SI SERVER puis Configuration SI Server ??????
Sélectionner LOCAL pour MODE DE LANCEMENT DES APPLICATIONS. Puis NOM DE DOMAINE COMPLET: Choisir ACTIVE
Entrer l’URL suivante dans le champ approprié: http://webos.deepidoo.com/beta/dj.ipk
Entrer le code à 4 chiffres donné par Deepidoo (nous contacter si vous ne l'avez, pas, il vous sera fourni instantanément), puis valider.
L’adresse du magasin sera alors affichée. Confirmer avec la touche 0 si l’adresse est exacte. NOTE: SI nécessaire, vous pouvez redémarrer la procédure avec la touche STOP.
