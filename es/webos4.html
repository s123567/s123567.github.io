Initialización

Elije el menú de inicio rápido

Elije tu idioma
Deja los 2 tipos de rotación de la pantalla en OFF
IMPORTANTE: Pon la pantalla a la hora correcta

Configuración de la red

Presiona el botón SETTINGS, y vayas al menú GENERAL, luego NETWORK

No podrá instalar la aplicación si su pantalla no está conectada a Internet.

No conecte su pantalla a un Wifi para invitados (este tipo de red no tiene suficientes permisos para permitir un funcionamiento óptimo)

Configura tu conexión a Internet, Ethernet o Wifi

Conexión Cableada
Si no está en una red con IP fija, se conectará automáticamente enchufando su cable
De lo contrario, por favor, rellena los datos de la red proporcionados por su oficina central o su proveedor de servicios de Internet

De lo contrario, presiona Editar, desmarca 'Definir automáticamente' e introduzca la información necesaria

Si seleccionas Wifi: Rellena la contraseña y conéctate

Instalación de la aplicación Deepidoo

Presiona el botón SETTING, vayas a EZ SETTINGS, luego IF SERVER SETTING

Elije SI SERVER SETTING

Ajuste los siguientes parámetros de la siguiente manera
=> Nombre de dominio calificado = ON
=> Ingresa esta URL 'http://webos.deepidoo.com/dj/ipk'
=> Application Launch Mode = LOCAL
=> Application Type = IPK

Haz clic en REMOTO y confirme

Reinicie la pantalla con el mando a distancia

Presiona el botón HOME, encuentra el icono del DJ y presiona OK.

Se le pedirá un código de 4 dígitos. Rellena el código dado por Deepidoo

Se indica la dirección de su sitio. Confirma pulsando la tecla 0, SÓLO si la dirección es correcta. Si no, llama a Deepidoo. NOTA: Si es necesario, puede reiniciar este paso pulsando el botón STOP
