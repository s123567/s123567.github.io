Choose the QUICK START menu

Choose your language.
Let BOTH screen rotations with the OFF setting.
IMPORTANT: Correctly setup the time.

Press the SETTINGS button (the button with the COG), and go to the GENERAL menu, then NETWORK

Setup the desired connection type (with cable or WIFI)

WITH CABLE
If you are not on a fixed IP network, you will get connected to the Internet automatically.
Otherwise, enter the network details provided by your internet provider or your Head Quarters.

WIFI: select the desired network and anter its passwordCONNEXION FILAIRE (RJ45)

Press the SETTING button, go to EZ SETTINGS and SI SERVER SETTING

Choose SI SERVER SETTING

Adjust the following parameters
- Fully Qualified Domain Name = ON.
- Enter this URL http://webos.deepidoo.com/dj/ipk
Be careful, it won't work if the URL is not correct
- Application Launch Mode = LOCAL
- Application Type = IPK

Click on REMOTE and Confirm, when asked

Restart your screen with the remote control

Push HOME button, find the DJ icon and press OK

You will be asked for a 4 digits number. Enter the number given by Deepidoo and validate ONLY if the address showing matches your shop's

Your shop address will be shown. Confirm by pressing 0, ONLY if it is correct. Otherwise, pease call Deepidoo. NOTE: IF needed, you can start over this process with the button STOP.
